An enumerable collection of UIDs of the mail messages to copy.

The name of the mailbox to copy the messages to.
mailbox (Optional)
Type: System..::..String
The mailbox the message will be copied from. If this parameter is omitted, the value of the DefaultMailbox property is used to determine the mailbox to operate on.

Remarks

When copying many messages, this method is more efficient than calling CopyMessage(UInt32, String, String) for each individual message.
