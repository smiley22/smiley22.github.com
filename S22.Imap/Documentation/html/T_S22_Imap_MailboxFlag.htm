MailboxFlag Enumeration
An IMAP mailbox can have a number of flags (read: attributes) set on it, indicating its designated purpose (such as "Spam" or "Drafts"). This is especially convenient if dealing with international mailbox names, because by examining the associated flags, the mailbox purpose can be easily determined, even if the mailbox name provides no useful indication.

Drafts2 The mailbox is used to hold draft messages, typically these are messages that are being composed but have not yet been sent.
Sent3 The mailbox is used to hold copies of messages that have been sent.
Spam4 The mailbox is where messages deemed to be junk mail (spam) are held.
Trash5 The mailbox is used to hold messages that have been deleted or marked for deletion.

Remarks

Unfortunately as with many things IMAP, reporting the flags described above is an optional extension (XLIST) to the IMAP protocol and as such may or may not be supported by an IMAP server.
